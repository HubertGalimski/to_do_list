# to_do_list

### The purpose of this project is to write a ToDo list using: 
- Java 11
- Spring Boot
- Hibernate
- Tomcat
- Maven
- Git 
- H2 Base

#### You can then access toDoList here:
http://localhost:8080/

#### Swagger REST API documentation presented here (after application start):
